# brute-force-attacks-project

🔐 Brute-Force Attacks – Project
By Inlighn Tech
This project simulates and analyzes brute-force attacks against vulnerable web applications (bWAPP/DVWA) using industry-standard penetration testing tools. It demonstrates how weak authentication systems can be exploited and teaches how to defend against such attacks.

Tools Used
Burp Suite – Intruder module with Cluster Bomb attack type
Hydra – Automated HTTP POST form brute-forcing

Labs Covered
Lab 1: Brute-force login using Burp Suite Intruder (Cluster Bomb)
Lab 2: Automated credential attack using Hydra on bWAPP login form

Deliverables
Full project report (.docx) with screenshots
Hydra command syntax for HTTP POST form attacks
Mitigation checklist to prevent brute-force attacks
